Image Editing Application

An application that allows its users to quickly make small adjustments to a large number of images. This program was developed for the Personnel Foundation of the Swedish postal service.

I developed a mass image editing program for the Personnel Foundation of the Swedish postal service (postnordplus.com). The objective of the program is to mass edit images that require changing the width and height, brightness, sharpness, vignette, vibrance, and much more.

The program includes two different modes: mass and individual. The first mode functionality is to mass edit all images at once with the selected parameters. While the other mode allows the user to select different parameters for the different images. The second mode also allows the user to select which part of the image should be cropped. The program enables the PostNordPlus employees to save a significant amount of time, specifically for images that will be uploaded to a website.

I was responsible to develop both the front- and the backend of this program, which includes functionalities for editing and working with images. I was also responsible for training the staff and on request supporting the end-users when needed.

Technologies Used

    The program was developed using Python with the library PyQt
    QT Designer
    OpenCV
